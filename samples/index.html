chartjs-plugin-doughnutlabel-v3 Sample Chart
Chart.js plugin for doughnut chart to display lines of text in the center

This Chart.js plugin module allows you to display multiple lines of dynamic text centered in the middle area of the Doughnut Chart - which can be customized for the font, color, size and other attributes. It can be downloaded as a package from https://www.npmjs.com/package/chartjs-plugin-doughnutlabel-v3
